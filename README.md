## 🌐 App URL
[MatchingBaseball](http://matchingbaseball.tokyo/)
## どんなアプリ？
練習や試合する時、人数が足りなく断念した経験ありますか？  
そんな時にこのアプリでプレイヤーを募集したり、応募できます。  
## 機能一覧
- ユーザー登録（編集）
- 画像アップロード
- 画像プレビュー
- 投稿機能（編集、削除）
- 投稿検索
- ダイレクトメッセージ機能
- メッセージ一覧表示
- 単体テスト
- レスポンシブデザイン
